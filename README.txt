sCRAPe
======

Search for data in android /dev files

1. Select the file from the drop down list. 
   /dev/exynos-mem might be interesting.

2. Enter a search phrase in the field below.
   "user", "username", etc.. 

3. Enter a maximum dump file size (MB). 
   Start small (5MB) to get a feel for how 
   long an operation taks.

4. Inspect search results or download the 
   dump file from the SD card.
